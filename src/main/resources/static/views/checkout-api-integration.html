MCS Click To Pay - Checkout API

Click To Pay Checkout

Perform Click to Pay Checkout
Important: This Checkout API call is not an independent step.
  • Before using this standalone API, go to the main dashboard of the Reference App and click on “Go to SDK Demo” under Click to Pay SDK Integration.
  • Step 1: Perform the init() call to initialize the SDK.
  • Step 2: Choose your flow:
      — New user: Fill out all required details (card information, billing address, and personal info), then perform checkout.
      — Returning user: Complete identity verification (via email or SMS).
  • After completing the flow above and reaching the final checkout step, select “No, will run perform standalone API call” when prompted.
  • Copy the required credentials (visible in the logs below the SDK demo) and use them in this Checkout API form.
Note: The SDK demo must be run first to get valid credentials. If you skip these steps, this API call will not work with dummy values.
